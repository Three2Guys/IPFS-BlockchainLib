# IPFS-BlockchainLib

## Installation

### Prerequisites
1. **Check Node.js and npm versions**
    ```sh
    node -v
    npm -v
    ```

### Installing Dependencies
2. **Install required npm packages**
    ```sh
    npm install --verbose
    ```

3. **Install Truffle globally**
    ```sh
    npm install -g truffle --verbose
    ```

### Setting Up Node Version (if needed)
4. **Install nvm (Node Version Manager)**
    ```sh
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.2/install.sh)"
    ```

5. **Use nvm to install and use Node.js version 18**
    ```sh
    nvm install 18
    nvm use 18
    ```

### Running the Project
6. **Start the development server**
    ```sh
    npm start
    ```
